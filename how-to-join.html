How to Join PeanutNetwork

Follow these simple steps to join our amazing Minecraft community!

1. Get Minecraft Java Edition

To join our server, you'll need the official Minecraft Java Edition. Unfortunately, we don't support Bedrock Edition or mobile versions.

If you don't have Minecraft Java Edition yet, you can purchase it from the official Minecraft website.

Make sure your game is updated to at least version 1.8 (we also support up to 1.21).

2. Launch Minecraft

Open the Minecraft Launcher on your computer and log in with your Mojang or Microsoft account.

Select the Java Edition from the versions list (make sure it's version 1.8 or higher).

Click "PLAY" to launch the game.

3. Add Our Server

Once in the game main menu:

  1. Click on "Multiplayer"
  2. Click on "Add Server"
  3. Enter a server name (like "PhantMC")
  4. Enter our server address:
play.phantwalker.com

No special ports or settings are needed. Just enter the address as shown above.

4. Join and Play!

After adding the server:

  1. Click "Done" to save the server
  2. Find "PhantMC" in your server list
  3. Double-click on it to connect

That's it! You'll spawn in our lobby where you can explore all the amazing features we offer.

If you have any trouble connecting, check our troubleshooting guide or join our Discord for help.
